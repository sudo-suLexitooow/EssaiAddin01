Guide pratique — Insérer une table des matières dans Word

Procédure objective et vérifiable pour obtenir une table automatique, numérotée et cliquable.

Objectif

Ce document explique, étape par étape, comment : créer des styles de titres, insérer une table des matières automatique (avec numéros de page et liens cliquables) et vérifier qu'elle est à jour.

Critères de validation

✓ Présence : un titre « Table des matières » suivi de la TOC
✓ Numéros de page : toutes les entrées montrent un numéro
✓ Modèle automatique : liens cliquables vers les titres
✓ À jour : reflète tous les titres du document

Étapes (Word — version récente)

  1. Préparer les titres
    • Sélectionner un paragraphe qui doit apparaître dans la TOC.
    • Appliquer Titre 1, Titre 2… via l’onglet Accueil.
  2. Placer le curseur sous le titre Table des matières.
  3. Insérer la table : Références → Table des matières → choisir un modèle automatique.
  4. Configurer les niveaux via Insérer une table des matières… si nécessaire.
  5. Mettre à jour : Références → Mettre à jour la table → Toute la table.
  6. Vérifier : liens cliquables + numéros corrects.

Raccourcis utiles

  • Ctrl + Entrée : saut de page.
  • Ctrl + A puis F9 : mise à jour des tables.
  • Modifier les styles TOC 1/TOC 2 pour personnaliser l'apparence.

Checklist finale

  1. Présence du titre « Table des matières ».
  2. Table issue d’un modèle automatique.
  3. Numéros de page visibles.
  4. Liens cliquables fonctionnels.
  5. Table mise à jour après modifications.
