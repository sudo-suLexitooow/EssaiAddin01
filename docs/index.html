Guide pratique — Insérer une table des matières dans Word

Procédure objective et vérifiable pour obtenir une table automatique, numérotée et cliquable.

Objectif

Ce document explique, étape par étape, comment : créer des styles de titres, insérer une table des matières automatique (avec numéros de page et liens cliquables) et vérifier qu'elle est à jour.

Critères de validation (liste vérifiable)

✓ Présence : un titre « Table des matières » suivi de la TOC
✓ Numéros de page : toutes les entrées montrent un numéro
✓ Modèle automatique : liens cliquables vers les titres
✓ À jour : reflète tous les titres du document

Étapes (exécution dans Microsoft Word — version récente)

  1. Préparer les titres
    • Sélectionner un paragraphe qui doit apparaître dans la TOC (ex. un titre de chapitre).
    • Dans l'onglet Accueil, appliquer un style de titre : Titre 1 pour les sections principales, Titre 2 pour les sous-sections, etc.
    • Répéter pour tous les titres du document.
  2. Placer le curseur
    • Positionner le curseur à l'endroit où la table des matières doit apparaître, sous le titre Table des matières (créez ce titre si nécessaire).
  3. Insérer la table des matières
    • Aller à Références > Table des matières.
    • Choisir un modèle automatique (ex. « Table automatique 1 ») — ces modèles créent des liens cliquables et affichent les numéros de page.
  4. Configurer les niveaux
    • Si certains styles de titres n'apparaissent pas, cliquer sur Table des matières > Insérer une table des matières... > Options — vérifier que Titre 1, Titre 2, ... sont mappés aux niveaux 1, 2, ...
  5. Mettre à jour
    • Après modification du document : Références > Mettre à jour la table > choisir Mettre à jour toute la table pour inclure nouveaux titres et numéros de page.
  6. Vérification finale
    • Cliquer sur quelques entrées de la TOC : elles doivent faire défiler le document vers le bon titre (lien cliquable).
    • Confirmer que tous les titres apparaissent et que les numéros correspondent aux pages.

Raccourcis utiles et conseils objectifs

  • Ctrl+Entrée : insérer un saut de page (utile pour séparer la TOC du reste).
  • Ctrl+A puis F9 : mettre à jour toutes les tables/champs (inclut la TOC).
  • Personnaliser l'apparence : modifier les styles TOC 1/TOC 2 via l'onglet Accueil > Styles > Gérer les styles.

Checklist finale (procédure de validation)

  1. Le document contient un titre exact : Table des matières.
  2. La table insérée est un modèle automatique (pas une table manuelle).
  3. Les numéros de page sont visibles pour chaque entrée.
  4. Les liens de la TOC sont cliquables et dirigent vers les titres.
  5. Après mise à jour (F9), la TOC reflète tous les titres présents.

Impression / Export

Pour fournir ce complément avec votre devoir : imprimez en PDF (Fichier > Enregistrer sous > PDF) — la TOC conserve les numéros mais les liens cliquables ne fonctionnent que dans le PDF si vous activez les signets/lien lors de l'export (option selon version).
